| metric | data_sources | calculation |
| --- | --- | --- |
| completion_rate | `course_enrollments` (learner-course assignment and enrollment status), `course_completions` (learner completion events), optional `course_catalog` (course activity/visibility filters) | `completion_rate = (count(distinct learner_id with completion_status = "completed" within reporting window) / count(distinct learner_id with enrollment_status in ["enrolled", "in_progress"] within the same scope)) * 100` |
| course_engagement_score | `learning_activity_events` (lesson views, video progress, quiz attempts, discussion actions), `session_logs` (session duration), `assessment_attempts` (attempt counts/scores) | Per learner-course-period, compute weighted normalized engagement: `engagement_score = 0.35 * norm(active_minutes) + 0.25 * norm(content_interactions) + 0.20 * norm(assessment_attempts) + 0.20 * norm(discussion_actions)`; then aggregate as average or percentile distribution by course/cohort. |
| drop_off_rate_by_stage | `learning_funnel_events` (enrolled, started, module milestones, completed), `course_structure` (module/stage order), optional `time_series_activity` (last activity timestamp) | For each funnel stage `s`: `drop_off_rate_s = ((learners_reached_s - learners_reached_next_stage) / learners_reached_s) * 100`; highlight dominant drop-off point as stage with max `drop_off_rate_s`; optionally segment by cohort, device, and tenant. |
