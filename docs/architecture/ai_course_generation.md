| pipeline_stage | inputs | outputs |
|---|---|---|
| Document ingestion | Source documents (PDF, DOCX, HTML, PPT, transcripts), metadata (course level, audience, language), ingestion rules (allowed file types, max size), OCR/ASR services for scanned docs or audio/video | Normalized text chunks, extracted structural elements (headings, tables, figures), source-to-chunk traceability map, cleaned metadata index |
| Topic extraction | Normalized chunks, course goals, target competency framework, domain taxonomy/ontology, stopword and deduplication rules | Ranked topic list, topic hierarchy (module \> lesson), key concepts/keywords per topic, prerequisite graph, confidence scores and supporting citations |
| Lesson generation | Topic hierarchy, prerequisite graph, pedagogical template (learning objectives, explanations, examples, activities), learner profile (level, pacing, locale), style constraints | Draft lessons per topic (title, objectives, explanation, worked examples, practice tasks, summary), estimated duration, reading level metrics, citation links to source chunks |
| Quiz generation | Generated lessons, topic key concepts, assessment policy (question mix, difficulty distribution, pass threshold), item templates (MCQ, T/F, short answer), answer rationale requirements | Quiz bank per lesson/module, answer keys with rationales, difficulty-tagged items, distractor quality checks, quiz metadata (coverage %, estimated completion time) |
