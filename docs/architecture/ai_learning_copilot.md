# AI Learning Assistant for LMS

| capability | input_data | output |
|---|---|---|
| concept explanation | competency framework, topic metadata, learner role, preferred language/reading level, prior knowledge signals | clear explanation of LMS concept with examples, analogies, and role-specific context; optional multilingual summary and key takeaways |
| learning guidance | learner goals, current proficiency scores, completed modules, time availability, deadlines, engagement history | personalized learning path with next-best lessons, weekly study plan, milestone targets, and adaptive sequencing based on progress |
| practice generation | target skill, difficulty level, question type preferences, past mistakes, curriculum alignment tags | auto-generated practice set (MCQs, scenarios, short answers), answer key, rationale, and dynamic hints tuned to weak areas |
| skill recommendations | assessment results, job-role requirements, competency gaps, performance trends, peer benchmarks | ranked skill recommendations with priority score, estimated effort, suggested resources, and expected impact on role readiness |
