1 — Path Details
Path title is required.
All Required
Every required node must be completed. Most common completion rule.
Required + Electives
Complete required nodes plus N electives. Elective count set after creation.
Milestone Based
Progress gates unlock sequentially as milestones are reached.
Score Threshold
Learner must achieve a minimum score across required nodes.
2 — Ownership & Audience
Owner is responsible for maintaining this path. Maps to ownerId (UUID) in CreateLearningPathRequest. Path owner is required.
Tags are sent as the audience object: { tag: true }. Press Enter or click + Add to add a tag.
3 — Curriculum
🗂️
Curriculum is configured after creation
Nodes (courses, assessments, milestones) and edges (prerequisites, sequencing) are added via PUT /learning-paths/{pathId}/nodes and PUT /learning-paths/{pathId}/edges after the path is created. The path ID is only available once the path has been saved.
Plan Your Curriculum (preview — not submitted)
No nodes planned yet. Drag courses and assessments here after creation to build your sequence.
Preview only — this data is not sent to the API. Add nodes after creation.
4 — Review & Publish
Save as Draft
Create the path as a draft. Add curriculum nodes before publishing.
Create & Publish
Create and immediately publish. POST /learning-paths then POST /learning-paths/{id}/publish.
Required by POST /publish — minimum 3 characters. Change reason is required (min 3 characters).
